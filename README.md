# SeleniumExplorers — Team UI Automation Project

Командный проект по автоматизации тестирования веб-приложения, разработанный в рамках курса QA Automation.

## 👥 Команда
Проект выполнен командой из 3 человек в ходе учебного курса по QA-автоматизации.

## 🛠 Технологии

- Java 17
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)
- Git / GitHub (совместная работа через Pull Requests)

## 📋 Что тестируется

Автоматизированное тестирование UI веб-приложения, включая:
- Авторизация и регистрация пользователей
- Навигация по страницам
- Проверка элементов интерфейса
- Валидация форм и обработка ошибок

## 📁 Структура проекта
src/
├── main/java/
│   └── pages/          # Page Object классы
└── test/java/
└── tests/          # Тестовые классы
## ▶️ Как запустить

1. Клонировать репозиторий:
```bash
git clone https://github.com/BermetB16/SeleniumExplorers.git
```

2. Открыть в IntelliJ IDEA

3. Запустить тесты:
```bash
mvn clean test
```

## 🤝 Командная работа

- Задачи распределялись между участниками
- Работа через Git: отдельные ветки, Pull Requests, code review
- Совместное планирование тест-кейсов
